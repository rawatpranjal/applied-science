# Applied Science

This repository contains working code organized into the following topics:

- **causal-inference**: Methods and tools for causal inference.
- **optimization**: Optimization techniques and algorithms.
- **recommendation**: Recommendation system implementations and concepts.
- **experimentation**: Experimental design and analysis.
- **machine-learning**: General machine learning models and tools.
- **reinforcement-learning**: Reinforcement learning theories and applications.
- **bayesian-tools**: Bayesian analysis methods and utilities.
- **time-series**: Time series analysis and forecasting techniques.
- **clustering**: Clustering algorithms and applications.
